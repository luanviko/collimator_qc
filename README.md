# Basic instructions

**Do not run all the scripts.**

In this order, run ONLY THESE scripts: 01a, 02c, 03, 07c, 08a. 

Only run the scripts below if you WANT to:

* Redo perspective and scaling parameters calculation: 02a, 02b, 02d, 02e.

* Calculate relative angles without laser correction: 04b. 

* Want to recalculate laser field parameters: 06a, 06b.  


Change the `DATA_DIR` path to wherever your RAW images are copied to. 

Change `PREPROCESSED_DIR` to where you prefer to keep stacked/ average images. 

Change `RUN_STRS` to whichever runs you want to analyze. 

Change the output folder convention to which one you prefer by modifying strings such as: `PLOT_DIR` and `FIT_DIR`.
